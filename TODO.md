## Language

Allow pull/push from any context to any other context. For example: use $sh to represent last bash output in any context. 

$pi pulls PI TOS to current context.

$rho pulls RHO panel content into current context.

$sh pulls BASH panel content into current context.

Similarly: writing to $pi $rho or $sh pushes to PI, RHO, BASH panels.
